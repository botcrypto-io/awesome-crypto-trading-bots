## Open Source Bots

- [Freqtrade](https://github.com/freqtrade/freqtrade) - Free, open source crypto trading bot written in Python.
- [Jesse](https://github.com/jesse-ai/jesse) - Advanced crypto trading framework.
- [Gekko](https://github.com/askmike/gekko) - Bitcoin trading bot written in Node.js.
- [Zenbot](https://github.com/carlos8f/zenbot) - Command-line bitcoin trading bot.
- [Hummingbot](https://github.com/hummingbot/hummingbot) - Open source market making and arbitrage bot.
- [Superalgos](https://github.com/Superalgos/Superalgos) - Free, open-source crypto trading bot platform.
- [Octobot](https://github.com/Drakkar-Software/OctoBot) - Open source crypto trading bot with web interface.
- [OpenClaw](https://github.com/openclaw/openclaw) - Open source self-hosted AI agent with crypto trading skills.

## Hosted & Managed Bots

- [WaliGPT Crypto Trading Bot](https://app.waligpt.com/explore) - Pre-configured OpenClaw agent for DEX token swaps, portfolio management, and wallet tracking. Deploy to an isolated cloud instance in one click — no setup required.
- [3Commas](https://3commas.io) - Cloud-based automated trading platform.
- [Cryptohopper](https://www.cryptohopper.com) - Cloud-based crypto trading bot.
- [Shrimpy](https://www.shrimpy.io) - Portfolio management and social trading platform.
- [Pionex](https://www.pionex.com) - Exchange with built-in trading bots.
- [Bitsgap](https://bitsgap.com) - AI-powered crypto trading platform.

## Frameworks & Libraries

- [CCXT](https://github.com/ccxt/ccxt) - CryptoCurrency eXchange Trading library supporting 100+ exchanges.
- [TA-Lib](https://ta-lib.org) - Technical analysis library.
- [Backtrader](https://github.com/mementum/backtrader) - Python backtesting library.
